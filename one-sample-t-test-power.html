Confidence Level

The probability that if the null hypothesis is true, we will correctly fail to reject it. It is equal to 1 - α (alpha). A 95% confidence level means there is a 5% chance of making a Type I error.

Power

The probability that if the alternative hypothesis is true, we will correctly reject the null hypothesis. It is equal to 1 - β (beta). This is the probability of detecting an effect if there is one.

Type I Error (α, alpha)

The error of rejecting a null hypothesis when it is actually true. This is a 'false positive'. The probability of a Type I error is denoted by alpha (α), which is the significance level of the test.

Type II Error (β, beta)

The error of failing to reject a null hypothesis when it is actually false. This is a 'false negative'. The probability of a Type II error is denoted by beta (β). Power is the complement of this (Power = 1 - β).
